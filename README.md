# 🐼 Pelu-xing (佩鲁星) - Backend & Database Infrastructure

Bienvenido al repositorio oficial del Backend y la Infraestructura de Base de Datos de **Pelu-xing**, nuestra tienda en línea especializada en peluches asiáticos. Este proyecto soporta la gestión del catálogo, control de inventario dinámico, procesamiento de órdenes y perfiles de usuario.

---

## 🧸 Descripción del Proyecto

Pelu-xing es una plataforma de e-commerce de alto rendimiento. Este repositorio contiene la API RESTful que sirve los datos al frontend, construida sobre un entorno robusto y escalable, diseñado para manejar picos de tráfico estacionales y proteger la información personal de nuestros clientes bajo estrictos estándares de seguridad.

---


## ⚙️ Requisitos Previos

Asegúrate de tener instalados los siguientes componentes antes de iniciar el entorno local:

- [Node.js](https://nodejs.org/) (v16 o superior)
- [MongoDB](https://www.mongodb.com/try/download/community) (Local o conexión a MongoDB Atlas)
- Git
